# Corporate Affairs Taxonomy Manager - Project Overview

## Executive Summary

A Next.js 16 web application for creating and managing hierarchical communication taxonomies. The application provides interactive tree and column views for navigating a strict 5-level hierarchy structure with export capabilities (JSON, CSV, PDF).

## Project Classification

| Attribute | Value |
|-----------|-------|
| **Repository Type** | Monolith |
| **Project Type** | Web Application |
| **Primary Language** | TypeScript |
| **Framework** | Next.js 16 (App Router) |
| **UI Library** | React 19 |
| **Styling** | Tailwind CSS 4 |
| **Data Storage** | File-based JSON |

## Purpose

The application serves Corporate Affairs teams who need to:
- Define and organize communication pillars and themes
- Create hierarchical taxonomy structures (5 levels deep)
- Assign audiences and geographies to taxonomy items
- Visualize relationships via tree diagrams or column navigation
- Export taxonomies for reporting and documentation

## Key Features

1. **Dual View Modes**
   - Tree View: Interactive flow diagram using @xyflow/react
   - Column View: Miller columns navigation (macOS Finder-style)

2. **5-Level Hierarchy**
   - Pillar → Narrative Theme → Subject → Topic → Subtopic
   - Each level has distinct colors and constraints
   - Objectives supported for top 3 levels only

3. **CRUD Operations**
   - Create, read, update, delete taxonomy nodes
   - Cascading deletes (removes all descendants)
   - Inline editing in detail panel

4. **Filtering & Search**
   - Filter by pillar
   - Search with ancestor/descendant inclusion
   - Real-time filtering

5. **Export Capabilities**
   - JSON (nested tree structure)
   - CSV (flat with hierarchy columns)
   - PDF (rendered tree diagram)

6. **Settings Management**
   - Configurable audiences list
   - Configurable geographies list

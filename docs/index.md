## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Key Features

1. **5-Level Taxonomy Hierarchy**
   - Pillar → Narrative Theme → Subject → Topic → Subtopic
   - Level-specific colors and constraints

2. **Dual View Modes**
   - Tree View: Interactive flow diagram
   - Column View: Miller columns navigation

3. **Full CRUD Operations**
   - Create, read, update, delete nodes
   - Cascading deletes for descendants

4. **Export Capabilities**
   - JSON (nested structure)
   - CSV (flat with hierarchy columns)
   - PDF (rendered tree diagram)

5. **Settings Management**
   - Configurable audiences
   - Configurable geographies
